## 1. Cox and Kings Travel Agency

The calculations for breakeven and target operating income rely on the following formulas:

- **Contribution Margin per Unit (CMU)** = Revenue per Ticket - Variable Cost per Ticket
    
- **Breakeven Units** = $\frac{\text{Fixed Costs}}{\text{CMU}}$
    
- **Target Income Units** = $\frac{\text{Fixed Costs} + \text{Target Operating Income}}{\text{CMU}}$
    

**Given:**

- Fixed Costs (FC) = Rs. 10,00,000 per month 11
    
- Target Operating Income (TOI) = Rs. 4,00,000 per month 2
    

---

- **CMU:** $\text{Rs. } 2,000 - \text{Rs. } 400 = \text{Rs. } 1,600$
    

(a) Breakeven Units: $\frac{\text{Rs. } 10,00,000}{\text{Rs. } 1,600} = \textbf{625 tickets}$

(b) Target Income Units: $\frac{\text{Rs. } 10,00,000 + \text{Rs. } 4,00,000}{\text{Rs. } 1,600} = \frac{\text{Rs. } 14,00,000}{\text{Rs. } 1,600} = \textbf{875 tickets}$
